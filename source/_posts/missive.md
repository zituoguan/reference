## 键盘快捷键

### 导航

| 快捷键        | 操作           |
| ------------- | -------------- |
| `Down`        | 下一项         |
| `Up`          | 上一项         |
| `Alt` `Up`    | 第一项         |
| `Alt` `Down`  | 最后一项       |
| `Right`       | 打开项         |
| `Left`        | 折叠项         |
| `Tab`         | 下一个范围     |
| `Shift` `Tab` | 上一个范围     |
| `PgUp`        | 向上滚动       |
| `PgDn`        | 向下滚动       |
