## 键盘快捷键

### 工具

| 快捷键    | 操作     |
| -------- | ---------- |
| `F`      | 画框      |
| `U`      | 圆角      |
| `T`      | 文本       |
| `S`      | 堆栈      |
| `L`      | 链接       |
| `Space`  | 平移画布 |

{.shortcuts}

### 图形

| 快捷键          | 操作    |
| ----------------- | --------- |
| `G`               | 图形   |
| `P`               | 路径      |
| `R`               | 矩形 |
| `O`               | 椭圆      |
| `V`               | 矢量    |
| `Cmd` `G`         | 编组     |
| `Cmd` `Shift` `G` | 取消编组   |

{.shortcuts}

| `Cmd` `Del`         | 移除包装器 |
