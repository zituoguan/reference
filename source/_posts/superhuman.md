## 键盘快捷键

### 操作

| 快捷键        | 操作                |
| ------------- | ------------------- |
| `Cmd` `K`     | Superhuman 命令     |
| `/`           | 搜索                |
| `Z`           | 撤销                |
| `?`           | 快捷键列表          |
| `X`           | 选择对话            |
| `Shift` `J/K` | 添加到选择          |

{.shortcuts}

### 筛选器

| 快捷键      | 操作      |
| ----------- | --------- |
| `Shift` `U` | 未读      |
| `Shift` `S` | 已加星标  |
| `Shift` `I` | 重要      |
| `Shift` `R` | 无回复    |

{.shortcuts}

### 邮件

| 快捷键      | 操作                          |
| ----------- | ----------------------------- |
| `C`         | 撰写                          |
| `Enter`     | 全部回复                      |
