## 键盘快捷键

### 所有页面

| 快捷键 | 操作                              |
| -------- | ----------------------------------- |
| `?`      | 显示键盘快捷键          |
| `[`      | 展开和折叠左侧导航 |
| `/`      | 聚焦站点搜索               |

{.shortcuts}

### 大多数页面（“您的工作”和“源代码”除外）

| 快捷键 | 操作                         |
| -------- | ------------------------------ |
| `.`      | 打开 Omnibar               |
| `J`      | 选择下一个项目               |
| `K`      | 选择上一个项目           |
| `Enter`  | 查看所选项目             |
| `G` `D`  | 前往“您的工作”仪表板      |
| `G` `A`  | 前往您的个人设置   |
| `Esc`    | 关闭对话框或移除焦点 |
| `U`      | 返回                        |

{.shortcuts}

### 仓库页面（“源代码”除外）

| 快捷键 | 操作                        |
| -------- | ----------------------------- |
| `C` `R`  | 创建仓库           |
| `I` `R`  | 导入仓库           |
| `R` `S`  | 打开仓库源代码        |
| `R` `C`  | 打开仓库提交       |
| `R` `B`  | 打开仓库分支      |
| `R` `P`  | 打开仓库拉取请求 |
| `R` `I`  | 打开仓库问题        |
| `R` `W`  | 打开仓库 Wiki          |
| `R` `D`  | 打开仓库下载     |
| `R` `A`  | 打开仓库设置      |
| `F`      | 搜索文件               |

{.shortcuts}

### 仓库页面（“源代码”和“设置”除外） {.row-span-2}

| 快捷键 | 操作                   |
| -------- | ------------------------ |
| `X` `F`  | 复刻仓库          |
| `X` `B`  | 创建分支            |
| `X` `C`  | 比较分支或标签 |
| `X` `P`  | 创建拉取请求      |
| `X` `I`  | 创建问题             |

{.shortcuts}

### 拉取请求 {.row-span-2}

| 快捷键       | 操作                                  |
| -------------- | --------------------------------------- |
| `Ctrl` `Enter` | 提交评论                        |
| `T` `C`        | 切换内联评论显示       |
| `P` `D`        | 切换到拉取请求“差异”选项卡     |
| `P` `C`        | 切换到拉取请求“提交”选项卡  |
| `P` `A`        | 切换到拉取请求“活动”选项卡 |
| `Shift` `T`    | 显示任务列表                  |

{.shortcuts}

### 带侧边栏的仓库页面

| 快捷键 | 操作                            |
| -------- | --------------------------------- |
| `]`      | 展开和折叠右侧边栏 |

{.shortcuts}

### 仓库源代码

| 快捷键 | 操作                |
| -------- | --------------------- |
| `F`      | 聚焦文件筛选器 |

{.shortcuts}
