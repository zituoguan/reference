## 键盘快捷键

### 移动

| 快捷键     | 操作         |
| ------------ | -------------- |
| `W`          | 前进   |
| `S`          | 后退      |
| `A`          | 左移      |
| `D`          | 右移     |
| `Left Shift` | 冲刺         |
| `Space`      | 跳跃           |
| `C`          | 切换蹲伏  |
| `Left Ctrl`  | 按住蹲伏 |

{.shortcuts}

| 快捷键 | 操作                    |
| -------- | ------------------------- |

| `Z`      | 终极技能          |
| `E`      | 互动/拾取        |
| `X`      | 备用互动        |
| `Tab`    | 切换物品栏          |
| `M`      | 切换地图                |
| `B`      | 切换开火模式          |
| `V`      | 近战                     |
| `R`      | 换弹                    |

| `G`      | 装备手榴弹             |
