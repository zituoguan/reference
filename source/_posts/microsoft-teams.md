## 键盘快捷键

### 常规

| 快捷键     | 操作                  |
| ---------- | ----------------------- |
| `Ctrl` `.` | 显示键盘快捷键          |
| `Ctrl` `E` | 转到搜索                |
| `Ctrl` `/` | 显示命令                |
| `Ctrl` `G` | 转到                    |
| `Ctrl` `N` | 开始新聊天              |
| `Ctrl` `,` | 打开设置                |
| `F1`       | 打开帮助                |
| `Esc`      | 关闭                    |
| `Ctrl` `=` | 放大                    |
| `Ctrl` `-` | 缩小                    |
| `Ctrl` `0` | 恢复默认缩放            |

{.shortcuts}

### 导航

| 快捷键                | 操作                     |
| --------------------- | -------------------------- |
| `Ctrl` `1`            | 打开活动                   |
| `Ctrl` `2`            | 打开聊天                   |
| `Ctrl` `3`            | 打开团队                   |
| `Ctrl` `4`            | 打开日历                   |
| `Ctrl` `5`            | 打开通话                   |
| `Ctrl` `6`            | 打开文件                   |
| `Alt` `Up`            | 转到上一个列表项           |
| `Alt` `Down`          | 转到下一个列表项           |
| `Ctrl` `Shift` `Up`   | 上移选定的团队             |
| `Ctrl` `Shift` `Down` | 下移选定的团队             |
| `Ctrl` `Shift` `F6`   | 转到上一节                 |
| `Ctrl` `F6`           | 转到下一节                 |

{.shortcuts}

### 消息

| `Ctrl` `O`         | 附加文件                          |
| `Shift` `Enter`    | 开始新的一行                      |
| `Alt` `Shift` `R`  | 回复话题                          |
