## 键盘快捷键

### 常规操作

| 快捷键                   | 功能                                              |
| ------------------------ | ------------------------------------------------- |
| `Ctrl` `S`               | 全部保存                                          |
| `Ctrl` `Alt` `Y`         | 同步                                              |
| `Ctrl` `Shift` `F12`     | 最小化或最大化编辑器                              |
| `Alt` `Shift` `F`        | 添加到收藏夹                                      |
| `Alt` `Shift` `I`        | 使用当前配置文件检查当前文件                      |
| `Ctrl` <code>\`</code>   | 快速切换方案                                      |
| `Ctrl` `Alt` `S`         | 打开设置对话框                                    |
| `Ctrl` `Alt` `Shift` `S` | 打开项目结构对话框                                |
| `Ctrl` `Tab`             | 在标签页和工具窗口间切换                          |

{.shortcuts}
