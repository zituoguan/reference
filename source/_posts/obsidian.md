## 键盘快捷键 {.cols-2}

### 默认快捷键

| 快捷键               | 操作                        |
| -------------------- | --------------------------- |
| `Ctrl` `S`           | 保存编辑器中的文件            |
| `Ctrl` `N`           | 创建新笔记                  |
| `Ctrl` `P`           | 打开命令面板                |
| `Ctrl` `O`           | 打开快速切换器              |
| `Ctrl` `Shift` `F`   | 搜索所有文件                |
| `Ctrl` `G`           | 打开关系图谱                |
| `Ctrl` `Alt` `Left`  | 后退                        |
| `Ctrl` `Alt` `Right` | 前进                        |
| `Ctrl` `F`           | 搜索当前文件                |
| `Ctrl` `E`           | 切换编辑或预览模式          |
| `Ctrl` `,`           | 打开设置                    |

{.shortcuts}

### 编辑

| 快捷键        | 操作                          |
| ------------- | ------------------------------- |
| `Ctrl` `B`    | 加粗选中文本                  |
| `Ctrl` `I`    | 斜体选中文本                  |
| `Ctrl` `K`    | 插入外部链接                  |
| `Ctrl` `]`    | 缩进                          |
| `Ctrl` `[`    | 取消缩进                      |
| `Alt` `Enter` | 跳转到选中的反向链接            |

{.shortcuts}
