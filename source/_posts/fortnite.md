## 键盘快捷键

### 主要控制

| 快捷键      | 动作                                    |
| ------------ | ----------------------------------------- |
| `R`          | 装弹                                      |
| `Q`          | 在战斗模式和建造模式之间切换                |
| `Space`      | 跳跃                                      |
| `Left Ctrl`  | 蹲伏                                      |
| `Left Shift` | 冲刺                                      |
| `=`          | 自动奔跑                                  |
| `W`          | 向前移动                                  |
| `S`          | 向后移动                                  |
| `A`          | 向左移动                                  |
| `D`          | 向右移动                                  |
| `M`          | 切换地图                                  |
| `Tab`        | 切换物品栏                                |
| `E`          | 使用或与对象互动                          |
| `F`          | 选择镐                                    |
| `1-5`        | 选择武器槽 1-5                            |
