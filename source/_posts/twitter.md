## 键盘快捷键

### 操作

| 快捷键         | 操作               |
| -------------- | ------------------ |
| `N`            | 新建推文           |
| `L`            | 喜欢推文           |
| `R`            | 回复推文           |
| `T`            | 转推               |
| `M`            | 私信               |
| `U`            | 静音账户           |
| `B`            | 屏蔽账户           |
| `Enter`        | 打开推文详情       |
| `O`            | 展开图片           |
| `/`            | 搜索               |
| `Ctrl` `Enter` | 发送推文           |

{.shortcuts}

### 导航

| 快捷键  | 操作               |
| ------- | ------------------ |
| `?`     | 完整键盘快捷键菜单 |
| `J`     | 下一条推文         |
| `K`     | 上一条推文         |
| `Space` | 向下翻页           |
| `.`     | 加载新推文         |

{.shortcuts}
