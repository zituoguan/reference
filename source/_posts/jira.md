## 键盘快捷键

### 全局快捷键

| 快捷键                       | 操作                           |
| ---------------------------- | -------------------------------- |
| `G` `D`                      | 转到仪表盘                     |
| `G` `P`                      | 浏览项目                       |
| `G` `A`                      | 转到敏捷看板                   |
| `G` `I`                      | 转到问题导航器                 |
| `G` `G`                      | 管理搜索对话框                 |
| `/`                          | 快速搜索                       |
| `C`                          | 创建问题                       |
| `?`                          | 打开快捷键帮助                 |
| `Ctrl` `Alt` `D`             | 仪表盘下拉菜单                 |
| `Ctrl` `Alt` `P`             | 项目下拉菜单                   |
| `Ctrl` `Alt` `I`             | 问题下拉菜单                   |
| `Ctrl` `Alt` `G`             | 敏捷看板下拉菜单               |
| `Ctrl` `Alt` `S`             | 提交表单                       |
| `Ctrl` `Alt` <code>\`</code> | 取消表单                       |
