## 键盘快捷键 {.cols-2}

### 常规操作 I

| 快捷键             | 操作                     |
| -------------------- | -------------------------- |
| `方向键` | 导航卡片             |
| `B`                  | 打开看板导航菜单    |
| `/`                  | 聚焦搜索框           |
| `C`                  | 归档卡片               |
| `D`                  | 截止日期                   |
| `E`                  | 快速编辑模式            |
| `Esc`                | 关闭菜单/取消编辑  |
| `Ctrl` `Enter`       | 保存文本                  |
| `Enter`              | 打开卡片                  |
| `F`                  | 打开卡片筛选菜单      |
| `L`                  | 标签                      |
| `;`                  | 切换标签名称显示         |
| `M`                  | 添加/移除成员         |
| `N`                  | 添加新卡片            |
| `,/./</>`            | 移动卡片到相邻列表 |
