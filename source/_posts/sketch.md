### 缩放和聚焦

| 快捷键                | 操作                             |
| ----------------------- | ---------------------------------- |
| `Cmd` `0`               | 缩放到 100%                       |
| `Cmd` `1`               | 缩放以适应画布上的所有元素     |
| `Cmd` `2`               | 缩放以适应所选图层(们)          |
| `Cmd` `3`               | 将所选图层(们)居中于画布 |
| `Shift` <code>\`</code> | 临时缩放到 100%       |
| `Cmd` `Ctrl` `F`        | 切换全屏                 |
| `Cmd` `Opt` `1`         | 显示或隐藏图层列表            |
| `Cmd` `Opt` `2`         | 显示或隐藏检查器             |
| `Cmd` `.`               | 进入演示模式            |
