## 键盘快捷键

### 编辑 {.row-span-2}

| 快捷键             | 操作                                           |
| ------------------ | ---------------------------------------------- |
| `Ctrl` `Alt` `I`   | 调整文档大小                                   |
| `Ctrl` `Alt` `C`   | 调整画布大小                                   |
| `.`                | 重置选择框                                     |
| `;`                | 切换吸附                                       |
| `Ctrl`             | 在当前图层上使用像素工具擦除                   |
| `X`                | 切换描边/填充颜色选择器                        |
| `Shift` `X`        | 交换描边/填充颜色选择器                        |
| `/`                | 在描边/填充颜色选择器上设置无填充              |
| `O`                | 切换裁剪工具叠加层                             |
| `Shift` `O`        | 翻转黄金螺旋叠加层 (裁剪工具启用时)            |
| `Esc`              | 取消调整大小、移动或创建操作                   |
| `(方向键)`         | 移动项目                                       |
| `Shift` `(方向键)` | 以10倍测量单位移动项目                         |
