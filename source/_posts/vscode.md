## Windows 键盘快捷键

### 常规

| 快捷键                | 操作                      |
| --------------------- | ------------------------- |
| `Ctrl` `Shift` `P`    | 显示命令面板              |
| `Ctrl` `P`            | 快速打开，转到文件...     |
| `Ctrl` `Shift` `N`    | 新建窗口/实例             |
| `Ctrl` `Shift` `W`    | 关闭窗口/实例             |
